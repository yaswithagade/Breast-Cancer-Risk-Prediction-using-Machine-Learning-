Breast Cancer Risk Classification using Machine Learning

This project uses the Breast Cancer Wisconsin Diagnostic Dataset to classify tumors as benign or malignant.

Technologies:
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

Models:
- Logistic Regression
- Random Forest
- Support Vector Machine

Evaluation Metrics:
- Accuracy
- Precision
- Recall
- F1 Score
